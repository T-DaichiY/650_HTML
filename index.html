GitHub Pagesで公開する最小構成

  1. このフォルダ内の index.html と4つのHTMLをGitHubリポジトリにアップロード。
  2. GitHubで Settings → Pages を開く。
  3. Build and deployment で Deploy from a branch を選ぶ。
  4. Branchを main、folderを /root にして保存。
  5. 数分後に https://ユーザー名.github.io/リポジトリ名/ で公開される。
